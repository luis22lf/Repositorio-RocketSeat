Comandos terminal

cd - navaga entre diretórios
mkdir - cria diretório (pasta)
toutch - cria arquivo
cp - copia arquivos (ultima entrada é o destino)(-r para copiar um diretorio)
mv - move arquivos (ultima entrada é o destino)(-r para mover um diretório)
rm - remove arquivos (remove definitivo)(-r para remover um diretório)(-f para forçar exclusão)
find - procura arquivos (. procura no diretório atual) (-type d(diretório) f(file)) (-name"NOME"(*d* procura tudo que tenha d no meio do nome *d procura tudo que comece com qualquer coisa e termine com d)
